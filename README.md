# ­ЪјБ рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃерЃў

рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃА рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃљрЃњрЃарЃЮрЃЋрЃћрЃЉрЃА рЃљрЃЏрЃўрЃюрЃЊрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃгрЃДрЃљрЃарЃЮрЃЊрЃљрЃю рЃЊрЃљ рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ.

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- ­ЪћЇ **рЃљрЃЊрЃњрЃўрЃџрЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃФрЃћрЃЉрЃюрЃљ** - рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃЮрЃџрЃЮрЃњрЃўрЃўрЃЌ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃўрЃЌ
- ­ЪЊі **рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў** - 1-рЃЊрЃљрЃю 10-рЃЏрЃЊрЃћ рЃерЃЎрЃљрЃџрЃўрЃЌ рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ

- ­ЪїЎ **рЃЏрЃЌрЃЋрЃљрЃарЃўрЃА рЃцрЃљрЃќрЃћрЃЉрЃў** - рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃљрЃќрЃћ рЃњрЃљрЃЋрЃџрЃћрЃюрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃЌ
- ­ЪЊЁ **5 рЃЊрЃдрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў** - рЃњрЃљрЃюрЃЋрЃарЃфрЃЮрЃЉрЃўрЃџрЃў рЃърЃћрЃарЃўрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА
- ­ЪЊ▒ **рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃЋ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў** - рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ рЃЊрЃљ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃќрЃћ
- ­ЪЄг­ЪЄф **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃћрЃюрЃљ** - рЃАрЃарЃБрЃџрЃў рЃџрЃЮрЃЎрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ
