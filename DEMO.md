# ­ЪјБ рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў - рЃЊрЃћрЃЏрЃЮ рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

## ­Ъџђ рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

### 1. рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
- рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `index.html` рЃцрЃљрЃўрЃџрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў
- рЃљрЃю рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў HTTP рЃАрЃћрЃарЃЋрЃћрЃарЃў:
  ```bash
  python -m http.server 8000
  ```

### 2. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### рЃФрЃћрЃЉрЃюрЃљ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЎрЃЮрЃЏрЃърЃџрЃўрЃбрЃў
1. **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў**: рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЎрЃарЃћрЃЉрЃљ - "рЃЌрЃЉрЃў", "рЃЉрЃљрЃЌ", "рЃЏрЃбрЃЎ"
2. **рЃџрЃЮрЃЎрЃљрЃфрЃўрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў**: рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃФрЃћрЃЉрЃюрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ - "рЃЦрЃљрЃџрЃљрЃЦрЃў", "рЃЏрЃЊрЃўрЃюрЃљрЃарЃћ", "рЃбрЃЉрЃљ"
3. **рЃАрЃгрЃарЃљрЃцрЃў рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃў**: рЃЊрЃљрЃљрЃгрЃћрЃЦрЃўрЃЌ рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃБрЃџ рЃџрЃЮрЃЎрЃљрЃфрЃўрЃћрЃЉрЃќрЃћ

#### рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃА рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ
- рЃџрЃЮрЃЎрЃљрЃфрЃўрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ:
  - РюЁ рЃЌрЃћрЃЋрЃќрЃљрЃЮрЃЉрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў (1-10 рЃАрЃЎрЃљрЃџрЃўрЃЌ)
  - РюЁ 8 рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃбрЃарЃўрЃЎрЃў рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃћрЃЉрЃўрЃЌ
  - РюЁ 5 рЃЊрЃдрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў

#### рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃЋ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃбрЃћрЃАрЃбрЃў
- **Desktop**: рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃћрЃбрЃарЃўрЃЎрЃў рЃЏрЃЕрЃЦрЃўрЃцрЃћрЃЊ рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў
- **Tablet**: 2-рЃЎрЃЮрЃџрЃЏрЃБрЃюрЃўрЃљрЃюрЃў grid
- **Mobile**: 1-рЃЎрЃЮрЃџрЃЏрЃБрЃюрЃўрЃљрЃюрЃў layout

## ­ЪљЏ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў (рЃЊрЃћрЃЏрЃЮ рЃЋрЃћрЃарЃАрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА)

2. **API рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ**: рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ
3. **рЃЮрЃцрЃџрЃљрЃўрЃю рЃарЃћрЃЪрЃўрЃЏрЃў**: рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА
4. **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ**: рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃў error handling
